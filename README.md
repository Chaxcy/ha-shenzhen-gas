# ha-shenzhen-gas

深圳燃气 Home Assistant 自定义集成，用于读取深圳燃气微信小程序接口中的燃气表、用气和账单数据。

## 功能

- 累计读数
- 今日用气量
- 昨日用气量
- 用气日期
- 余额
- 上期账单金额
- 上期用气量
- 阶梯气费
- 上期账单周期
- 下一账单日
- 最新读数
- 最近更新
- 阀门状态
- 电池电量
- 在线状态

集成类型为 `cloud_polling`，默认每 30 分钟刷新一次数据。

## 安装

### HACS

1. 打开 HACS。
2. 进入“集成”。
3. 点击右上角菜单，选择“自定义存储库”。
4. 添加仓库：

   ```text
   https://github.com/Chaxcy/ha-shenzhen-gas
   ```

5. 类别选择“集成”。
6. 搜索并安装“深圳燃气”。
7. 重启 Home Assistant。

### 手动安装

1. 下载本仓库。
2. 将 `custom_components/shenzhen_gas` 复制到 Home Assistant 的 `custom_components` 目录。
3. 重启 Home Assistant。

目录结构应类似：

```text
config/
  custom_components/
    shenzhen_gas/
      manifest.json
      sensor.py
      ...
```

## 配置

重启后，在 Home Assistant 中进入：

```text
设置 -> 设备与服务 -> 添加集成 -> 深圳燃气
```

需要填写：

| 字段 | 说明 |
| --- | --- |
| 客户号 | 深圳燃气客户号 |
| 燃气表号 | 物联网燃气表号 |
| CodeId | 深圳燃气微信小程序请求头中的 `CodeId` |

`CodeId` 获取方式见：[获取 CodeId 教程](docs/get-codeid.md)。

## CodeId 说明

深圳燃气部分业务接口会校验微信小程序请求头里的 `CodeId`。实测不带 `CodeId` 时，账单接口会返回缺少参数或页面加载失败。

`CodeId` 通常长期不变，只需要获取一次。更换微信号、重新绑定账户或深圳燃气后端调整时，可能需要重新获取。

请勿公开自己的 `CodeId`、客户号、燃气表号或抓包文件。

## 常见问题

### 连接失败

请检查：

- 客户号是否正确
- 燃气表号是否正确
- `CodeId` 是否完整复制
- Home Assistant 是否可以访问 `wechat.szgas.com.cn`

### 部分实体为空

可能原因：

- 当前账户没有对应数据
- 账单尚未出账
- 深圳燃气接口临时异常
- 表具没有上报最新数据

### CodeId 会过期吗？

`CodeId` 不像接口 `access_token` 那样频繁过期。集成会自动刷新 `access_token`，但如果深圳燃气返回未登录、缺少参数或页面加载失败，可以尝试重新获取 `CodeId`。
